# CLAUDE.md 초기화

프로젝트에서 Claude Code를 효과적으로 사용하기 위한 운영 지침 CLAUDE.md를 생성합니다.

## 사용법

```
/init-claude
```

---

## 프로세스

### 1단계: 기존 파일 확인

- CLAUDE.md가 이미 존재하면 덮어쓸지 확인
- 존재하지 않으면 바로 생성

### 2단계: 기본 템플릿 생성

아래 템플릿으로 CLAUDE.md 생성:

```markdown
# CLAUDE.md

## 문서 관리 규칙

- 이 파일은 500자 이내 유지 (최대 1500자 초과 금지)
- 새 규칙 추가 시 오래된 규칙 정리
- 프로젝트 진행하며 학습한 패턴만 기록

## 코드 원칙

- 간결하게: 최소한의 코드로 목적 달성
- 재사용 우선: 기존 것 활용, 없으면 재사용 가능하게 작성
- 중복 금지: 비슷한 코드 반복되면 추상화

## 작업 규칙

- 변경 전 기존 코드 먼저 읽기
- 요청받은 범위만 수정, 과도한 리팩토링 금지

## 코드 스타일

- (프로젝트 진행하며 추가)
```

### 3단계: 사용자 확인

> 기본 CLAUDE.md를 생성했습니다.
> 추가할 프로젝트별 규칙이 있나요?

피드백 있으면 반영.

---

## 핵심 원칙

1. **짧게 유지**: 길어지면 읽히지 않음
2. **실용적 규칙만**: 실제로 필요했던 것만 추가
3. **진화하는 문서**: 처음엔 최소한, 작업하며 보강
