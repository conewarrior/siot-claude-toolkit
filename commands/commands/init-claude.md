  - 기존 내용은 절대 삭제하지 않음
- 존재하지 않으면 템플릿으로 새로 생성

### 2단계: 기본 템플릿

아래 템플릿으로 CLAUDE.md 생성:

```markdown

- 프로젝트 진행하며 학습한 패턴만 기록

## PRD/기획 작성 가이드

새 기능 기획 시 아래 항목을 사용자에게 질문하며 구체화:

- 목표: 이 기능으로 해결하려는 문제는?
- 대상 사용자: 누가 사용하는가?
- 핵심 기능: 반드시 있어야 할 기능은?
- 제약 조건: 기술적/시간적 제약이 있는가?
- 성공 기준: 어떻게 되면 성공인가?

## 코드 원칙

- 간결하게: 최소한의 코드로 목적 달성
- 재사용 우선: 기존 것 활용, 없으면 재사용 가능하게 작성
- 중복 금지: 비슷한 코드 반복되면 추상화

## 작업 규칙

- 변경 전 기존 코드 먼저 읽기
- 요청받은 범위만 수정, 과도한 리팩토링 금지

## 개발 컨벤션
```
